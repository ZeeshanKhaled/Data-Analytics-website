## Project Overview

This is a multi-page data analytics website designed to explain what data analytics is, how it can support businesses, and what tools or services the company provides. The site presents data analytics through a simple, accessible layout with pages for information, resources, mathematical tools, references, and contact details.

## Website Description

Data Analytics is an informational business-style website that introduces users to the value of data analysis, visualisation, and predictive insights. It includes educational content, service information, mathematical tool examples, references, and a contact page for users who want further support.

## Pages

| Page | File | Purpose |
|---|---|---|
| Home | `index.html` | Introduces data analytics using a detective-style explanation and highlights example business improvements. |
| News and Resources | `news.html` | Shares recent trends, research, insights, and source links. |
| Mathematical Tools | `tools.html` | Presents chart and data visualisation tools such as bar charts, donut charts, pie charts, trends, and bubble graphs. |
| About | `about.html` | Explains who the company is, what services it offers, and who it has helped. |
| References | `references.html` | Lists sources used to support the website content. |
| Contact Us | `contact.html` | Provides contact details and a basic contact form layout. |

## How to Run Locally

1. Download or clone the repository.
2. Keep the HTML, CSS, JavaScript, and image files in their correct folders.
3. Open `index.html` in a web browser.

For easier testing while editing, use the **Live Server** extension in Visual Studio Code.

## Deployment

The website is hosted using GitHub Pages:

https://zeeshankhaled.github.io/Data-Analytics-website/

To deploy updates:

1. Push the latest project files to the GitHub repository.
2. Go to the repository settings.
3. Open the **Pages** section.
4. Select the correct branch, usually `main`.
5. Save and wait for GitHub Pages to publish the updated site.
